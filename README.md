# Optical Surface Analyzer

Desktop application for analyzing optical surfaces built with Electron and React.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

## Development

```bash
npm run dev
```

## Building

```bash
npm run build
```

## Project Structure

```
SE3/
├── src/
│   ├── main.js          # Electron main process
│   ├── preload.js       # Preload script for IPC
│   ├── renderer.js      # React application
│   ├── index.html       # HTML template
│   └── styles.css       # Styles
├── package.json
└── README.md
```
